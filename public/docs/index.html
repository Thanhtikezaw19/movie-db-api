Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Auth Management

APIs to manage the Auth resource

POST api/register

Example request:
curl --request POST \
    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"gstnchhfigqarazflkyorzird\",
    \"email\": \"crooks.barney@example.org\",
    \"password\": \"yboCNBCa)\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "gstnchhfigqarazflkyorzird",
    "email": "crooks.barney@example.org",
    "password": "yboCNBCa)"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

POST api/movies

Example request:
curl --request POST \
    "http://localhost/api/movies" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "title=ad"\
    --form "summary=voluptas"\
    --form "genres=iusto"\
    --form "author=ipsa"\
    --form "tags=eos"\
    --form "imdb_rating=4"\
    --form "cover_image=@C:\Users\ThanHtikeZaw\AppData\Local\Temp\php17C6.tmp" 
const url = new URL(
    "http://localhost/api/movies"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('title', 'ad');
body.append('summary', 'voluptas');
body.append('genres', 'iusto');
body.append('author', 'ipsa');
body.append('tags', 'eos');
body.append('imdb_rating', '4');
body.append('cover_image', document.querySelector('input[name="cover_image"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
